# Snake with JavaFX


### Keyboard shortcuts
* Arrows → Move
* Esc → Exit
* R → Reset game
* F11 → Full Screen
* Space   → Increase snake speed / food price
* Shift → Decrease snake speed / food price

##### Change tail color
* 1 → Green
* 2 → YellowGreen
* 3 → Orange
* 4 → Blue
* 5 → Black
* 6 → Random
